## Installation & Usage

1. Download the latest AppImage from the [Releases](https://github.com/tahsinzidane/emoji-verse/releases).

2. Make it executable:

```bash
chmod +x emoji-verse-*.AppImage
````

3. Run it:

```bash
./emoji-verse-*.AppImage
```
